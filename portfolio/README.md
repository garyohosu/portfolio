# hantani - AI時代の創作研究者ポートフォリオサイト

AI時代の創作研究者・技術系ライター・元組み込みプログラマー「hantani」のポートフォリオサイトです。Creative Vibe Writing（CVW）研究の先駆者として、人間とAIの協働による新しい創作手法を研究・実践しています。

## 🎯 プロジェクト概要

このサイトは、技術と文学を橋渡しする新時代のクリエイターとしての活動を紹介するポートフォリオサイトです。モダンで洗練されたデザインと、スムーズなユーザー体験を提供します。

## ✨ 主な機能

### 完成済み機能

- ✅ **レスポンシブデザイン**: スマートフォン・タブレット・デスクトップに完全対応
- ✅ **ヒーローセクション**: 印象的なグラデーション背景とアニメーション
- ✅ **プロフィールセクション**: 詳細な自己紹介と専門分野の紹介
- ✅ **スキルセクション**: 4つの主要スキルカテゴリの表示
- ✅ **実績セクション**: 統計データと主要な成果の表示
- ✅ **執筆活動・SNSセクション**: 各プラットフォームへのリンク
- ✅ **スムーズスクロール**: セクション間のナビゲーション
- ✅ **ハンバーガーメニュー**: モバイル対応のナビゲーション
- ✅ **スクロールアニメーション**: 要素の表示時のフェードインアニメーション
- ✅ **カードホバーエフェクト**: 3Dパララックス効果
- ✅ **数字カウントアップアニメーション**: 実績セクションの統計表示
- ✅ **アクティブセクションハイライト**: 現在表示中のセクションをナビゲーションで表示

## 📝 カスタマイズガイド

### 色の変更

`css/style.css`の`:root`セクションで色変数を変更できます：

```css
:root {
    --primary-color: #2563eb;  /* メインカラー */
    --accent-color: #8b5cf6;   /* アクセントカラー */
    /* その他の色変数 */
}
```

### コンテンツの更新

`index.html`で各セクションのテキストを直接編集できます。

### アニメーションの調整

`js/main.js`でアニメーションのタイミングや効果を調整できます。
